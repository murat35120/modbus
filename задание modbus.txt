Функции принимают параметры, они передаются в объекте POST запросов

API работы с приложением
Управляется  POST запросами
Запрос JSON строка объекта
{
  api:api,
  cmd:cmd,
  data:{
      name1:value1,
     ......
   }
}
api - название объекта хранения команд (converter, controller, server, external_server)
cmd - название команды
data - набор отправляемых данных
Ответ содержит, данные из запроса   api,  cmd и данные ответа
{
  api:api,
  cmd:cmd,
  ansver:{
      name1:value1,
     ......
   }
}
